
<h2>Java application and Spring Boot.</h2>

<h3>Clone project</h3>

```
git clone https://github.com/leo-demetrio/api-springboot-postgres

```
After clone project and executing, just open the following address and view the project execution:
```
http://localhost:8080/
/users
/orders
/categories
/products

```
